# 🖌️ 杀戮尖塔2 - 数字琥珀自动画板 (Slay the Spire 2 Painter)

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

专门为《杀戮尖塔2》(Slay the Spire 2) 打造的全自动右键连笔画图辅助工具。
采用独特的**“数字琥珀”**全屏冻结选区机制，支持将本地图片、自定义文字一键转化为纯净线稿，并在游戏内以极快的速度自动绘制。

> **“让数字世界的瞬间凝固成琥珀，再用代码将其拓印在尖塔的高墙之上。”**

---

## ✨ 核心特性

* 🧊 **“数字琥珀”沉浸式选区**：点击开始后，瞬间冻结当前屏幕画面并进入暗场模式。通过鼠标左键框选，实现所见即所得的绝对比例等比映射绘制。
* 🖼️ **双引擎线稿提取**：
* **图片转线稿**：内置 OpenCV 高斯模糊去噪与自适应 Canny 边缘检测，支持实时调节线稿精细度（1-10档）。
* **文字转线稿**：支持调用 Windows 系统原生字体库，输入文字后自动计算包围盒（Bounding Box），生成自适应比例的完美线稿。


* ⚡ **狂暴连笔绘制算法**：摒弃低效的逐像素点按，采用寻边轮廓算法（FindContours）模拟真实人类连笔轨迹。支持无级调节跳帧步长，最快可瞬间刷完一整面墙。
* 🛡️ **底层硬件级模拟**：使用 Windows `ctypes.windll.user32` 发送底层鼠标事件，完美绕过游戏引擎的输入屏蔽。
* 🛑 **绝对安全急停机制**：全局监听键盘，绘制过程中如遇突发情况，**随时按下 `P` 键**即可瞬间强制中断绘制并抬起鼠标右键。
* 📺 **实时分屏预览**：右侧自带巨大的实时渲染面板，参数调整所见即所得。

## 🚀 快速使用 (小白玩家必看)

无需安装任何环境，直接下载打包好的独立程序即可使用：

1. 前往本仓库的 [Releases 页面](https://www.google.com/search?q=../../releases) 下载最新版本的 `spire_painter.exe`。
2. ⚠️ **极其重要**：右键点击 `spire_painter.exe`，选择 **“以管理员身份运行”**（否则全局 P 键急停和底层鼠标模拟将失效）。
3. 在软件左侧选择“外部图片”或“输入文字”，生成线稿。
4. 调整好“精细度”与“绘制速度”。
5. 切入《杀戮尖塔2》游戏内，找到可以右键作画的墙壁/界面。
6. 点击软件底部的 **“🚀 开始绘制”**，屏幕将变暗冻结。
7. 用鼠标左键在屏幕上框选你要作画的区域，松开鼠标，奇迹就会开始！

> **急停提示**：如果画歪了或者想中止，请疯狂敲击键盘上的 **`P`** 键！

## 💻 开发者指南 (从源码运行)

如果你想二次开发或自己打包，请遵循以下步骤：

### 1. 克隆仓库

```bash
git clone https://github.com/你的用户名/SpirePainter2.git
cd SpirePainter2

```

### 2. 安装依赖

强烈建议使用 Python 3.8+ 版本：

```bash
pip install opencv-python numpy Pillow keyboard

```

*(注意：GUI 界面使用 Python 内置的 Tkinter 构建，无需额外安装)*

### 3. 运行脚本

由于代码中包含全局键盘钩子，**请确保你的终端/命令行是以管理员权限启动的**：

```bash
python spire_painter.py

```

### 4. 重新打包为 EXE

如果修改了源码需要重新打包：

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --icon=brush.ico spire_painter.py

```

生成的 `.exe` 文件将出现在 `dist` 文件夹中。

## ⚠️ 常见问题 (FAQ)

**Q1：为什么运行 `.exe` 没反应，或者按 P 键停不下来？**
**A**：Windows 的安全机制限制了后台键盘监听。请务必右键 `.exe` -> **以管理员身份运行**。

**Q2：杀毒软件报毒/自动删除了软件？**
**A**：由于软件包含了“全屏截取”、“模拟鼠标操作”以及“全局键盘监听”这三个敏感行为，部分杀毒软件（如 360、Windows Defender）会产生误报。本软件完全开源，绝对安全，请在杀毒软件中添加信任或白名单即可。

**Q3：游戏里画出来的线断断续续的，像虚线？**
**A**：说明你设置的“绘制速度（步长）”太快，超过了游戏引擎的捕获帧率。请在左侧将速度滑块调低（建议 2~4），让线条更平滑。

**Q4：照片转出来的线稿太乱了，导致画不完？**
**A**：请在“方案A”中，将“线稿精细度”滑块调低（例如调到 2 或 3），然后点击“刷新线稿”，直到右侧预览区的画面变得简洁、抽象。
